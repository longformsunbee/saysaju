# 🌙 사이사주 HTML 버전 - 실행 방법

## ✨ 간단한 실행 방법 (3가지)

### 방법 1: VS Code Live Server (가장 추천!) ⭐

1. **VS Code 설치**
   - https://code.visualstudio.com/ 에서 다운로드

2. **Live Server 확장 설치**
   - VS Code 열기
   - 왼쪽 확장(Extensions) 아이콘 클릭 (Ctrl+Shift+X)
   - "Live Server" 검색
   - "Live Server" by Ritwick Dey 설치

3. **실행**
   - VS Code에서 프로젝트 폴더 열기
   - `index.html` 파일 우클릭
   - **"Open with Live Server"** 클릭
   - 자동으로 브라우저가 열립니다!

---

### 방법 2: 브라우저에서 직접 열기 (가장 쉬움!)

1. **파일 탐색기에서 `index.html` 파일을 찾기**
2. **더블클릭** 또는 **우클릭 → 연결 프로그램 → 브라우저 선택**
3. 끝! 바로 볼 수 있습니다 🎉

**주의:** 일부 기능(예: 링크 복사)은 로컬 파일에서 제한될 수 있습니다.

---

### 방법 3: Python 간단 서버 (Python 설치 시)

Python이 설치되어 있다면:

```bash
# 프로젝트 폴더에서 실행
python -m http.server 8000
```

그다음 브라우저에서 `http://localhost:8000` 열기

## 🔧 문제 해결

### 아이콘이 안 보여요
- 인터넷 연결을 확인하세요 (CDN 사용)

### 스타일이 이상해요
- `styles.css` 파일이 같은 폴더에 있는지 확인
- 브라우저 캐시를 삭제하고 새로고침 (Ctrl+F5)

### 기능이 작동하지 않아요
- `script.js` 파일이 같은 폴더에 있는지 확인
- 브라우저 콘솔(F12)에서 에러 확인
