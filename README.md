# 🌙 사이사주 - 토정비결·사주풀이 전문 웹사이트

따뜻하고 신비로운 분위기의 토정비결·사주풀이 서비스입니다.

## ✨ 주요 기능

- 📅 생년월일 기반 운세 분석
- 🌟 오늘의 운세, 올해의 흐름, 사주 분석
- 🎨 네이비 + 골드 테마의 고급스러운 디자인
- 📱 모바일/PC 반응형 지원
- ⚡ 부드러운 애니메이션 효과
- 💫 **Node.js 설치 불필요!** 순수 HTML로 바로 실행

## 🛠 기술 스택

- **HTML5** - 시맨틱 마크업
- **CSS3** - 커스텀 애니메이션
- **JavaScript (ES6+)** - 순수 바닐라 JS
- **Tailwind CSS** - CDN 방식 (설치 불필요)
- **Lucide Icons** - CDN 방식 (설치 불필요)
- **Fonts**: Nanum Myeongjo, Pretendard

## 🚀 초간단 실행 방법

### 방법 1: 더블클릭! (가장 쉬움) ⭐

```
1. index.html 파일을 찾기
2. 더블클릭!
3. 끝! 🎉
```

### 방법 2: VS Code Live Server (추천!)

```
1. VS Code에서 프로젝트 열기
2. Live Server 확장 설치
3. index.html 우클릭 → "Open with Live Server"
4. 끝! 🎉
```

**설치나 빌드 과정이 전혀 필요 없습니다!**

자세한 실행 방법은 [HTML_실행방법.md](HTML_실행방법.md)를 참고하세요.
